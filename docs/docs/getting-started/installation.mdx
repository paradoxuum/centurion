# Installation

## roblox-ts

Commander requires **roblox-ts**. An installation guide can be
found in its [documentation](https://roblox-ts.com/docs/setup-guide).

## Commander

Commander is not yet available through the npm registry.

In order to install it, you will have to use the GitHub URL.

<Tabs>
    <TabItem value="npm" default>
    ```bash title="Terminal"
    npm install paradoxuum/commander
    ```
    </TabItem>
	<TabItem value="yarn">
    ```bash title="Terminal"
    yarn add https://github.com/paradoxuum/commander.git
    ```
    </TabItem>
	<TabItem value="pnpm">
    ```bash title="Terminal"
    pnpm add paradoxuum/commander
    ```
    </TabItem>
</Tabs>
