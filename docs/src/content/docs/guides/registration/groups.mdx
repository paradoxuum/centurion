Groups must be registered through the [registerGroup](/reference/registry#registergroupgroups) method.

Similar to commands, groups are synchronized to the client.

<Code lang="ts" code={`
CenturionServer.start((registry) => {
	registry.registerGroup(
		{
			name: "info",
			description: "View info about a user or the server",
		},
		{
			name: "user",
			description: "View info about a user",
			parent: ["info"],
		},
		{
			name: "server",
			description: "View info about the server",
			parent: ["info"],
		},
	);
}).catch((err) => warn("Centurion could not be started:", tostring(err)));
`} />
